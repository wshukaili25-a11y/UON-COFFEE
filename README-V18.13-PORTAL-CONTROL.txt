# UON Hub V18.13 — Portal Control

تم تنفيذ:
- إعادة تصميم واجهة الصفحة الرئيسية كسلايدر تلقائي.
- إدارة الشرائح بالكامل من بوت تيليجرام: إضافة، تعديل، إخفاء، ترتيب، حذف.
- روابط الجامعة المهمة أسفل الصفحة الرئيسية، وتدار من قسم المواقع المهمة في البوت.
- إضافة Moodle وEduWave والأعذار الطبية وغير الطبية والتقويم الرسمي.
- إضافة الروابط نفسها كأزرار سريعة داخل UON AI.
- ربط صفحة التقويم بالمصدر الرسمي لجامعة نزوى، مع إبقاء الأحداث المحلية.
- حالة مستقلة لقسم مراكز الدعم: تشغيل/إيقاف/صيانة/قريبًا.
- التحكم بحقوق الموقع ووصف الفوتر من تيليجرام.

## النشر
1. نفذ UON_HUB_V18_13_PORTAL_CONTROL.sql في Supabase SQL Editor.
2. ارفع ملفات الموقع إلى GitHub/Vercel.
3. انشر الدالتين:
   supabase functions deploy telegram-admin --project-ref irkhvydgxpseflggbeqq --no-verify-jwt
   supabase functions deploy uon-ai --project-ref irkhvydgxpseflggbeqq --no-verify-jwt

ملاحظة: رابطا الأعذار مبدئيًا يشيران إلى صفحة السياسة الرسمية؛ يمكن استبدالهما بالرابطين المباشرين من تيليجرام متى توفرا.
