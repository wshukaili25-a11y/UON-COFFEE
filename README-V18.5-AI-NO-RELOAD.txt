- صفحة AI كانت تعيد التحميل عند الضغط على إرسال.
- منع سلوك النموذج الافتراضي داخل HTML حتى لو لم يحمل JavaScript.
- تركيب حدث الإرسال قبل فحص الصيانة وقبل أي طلب Supabase.

- فشل فحص الصيانة أو حالة الأداة لم يعد يمنع المحادثة من العمل.

النشر:
1. ارفع جميع الملفات.
2. لا يوجد SQL جديد.
